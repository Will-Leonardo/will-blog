## JS中string数组和number数组之间快速互转

```js
 let x1 = ['1','2'];
 console.log(x1);
 //["1", "2"]

 x1 = x1.map(Number);
 console.log(x1);
 //[1, 2]
```

```js
let x1 = [1,2];
console.log(x1);
//[1, 2]

x1 = x1.map(String);
console.log(x1);
//["1", "2"]
```
