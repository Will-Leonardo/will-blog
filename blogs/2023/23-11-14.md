## font-display: swap; 

Ensure text remains visible during webfont loading. 
能确保网络字体加载完成前，使用浏览器的默认字体，能提高页面性能

```css
@font-face {
  font-family: 'MyWebFont';
  src: url('webfont.eot');
  font-display: swap;
}
```


## 静态资源使用长时间的缓存

配置您的服务器以返回Cache-ControlHTTP 响应标头：

```
Cache-Control: max-age=31536000
```
