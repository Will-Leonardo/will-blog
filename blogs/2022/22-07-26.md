## ES6 解构赋值踩坑

1. let { a = 1 }  = {  a : null  }        //   a = null
2. let { a = 1 }  = {  a : undefined  }   //   a = 1

有时候还是老老实实 写  ```a = ojb.a ?? 1 ``` 吧

:::details ?? or ||

?? 能过过滤掉 null 和 undefined 

|| 会将第一个值转换成boolean类型，为true取第一个值，为false取第二个值

```
// ??
undefined ?? 2	// 2
null ?? 2		// 2
0 ?? 2			// 0
"" ?? 2			// ""
true ?? 2		// true
false ?? 2		// false

// ||
undefined || 2	// 2
null || 2		// 2
0 || 2			// 2
"" || 2			// 2
true || 2		// true
false || 2		// 2
```

**总的来说，是否有值场景使用??更适合**
:::
